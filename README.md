# Site-Zaban
Site Zaban Azar

This project uses vanilla JavaScript to create an interactive and modern landing page experience without relying on external libraries.


A modern and lightweight **spotlight cursor effect** built with vanilla HTML, CSS, and JavaScript.  
This project creates a smooth glowing light that follows the mouse movement, inspired by Apple-style UI interactions.

## 🎯 Features

* 🍔 Responsive mobile navigation with a smooth hamburger menu toggle.
* ✨ Scroll reveal animations powered by the Intersection Observer API for better performance.
* 🌌 Dynamic hero background with randomly generated animated stars.
* 🔤 Floating letter particles that create a unique visual effect in the hero section.
* ⚡ Lightweight, fast, and dependency-free implementation using pure JavaScript.
* ♿ Improved accessibility with dynamic  updates for the mobile navigation button.
* 🚀 Optimized animations that trigger only when elements enter the viewport, reducing unnecessary rendering and improving performance.
